### install

clone the repo:

```
git clone https://github.com/MonikaaS/real-time-web-1920.git
```

navigate to the repo

```
cd real-time-web-1920
```

install

```
npm install
```

run

```
npm run dev [to run nodemon]
or
npm start
```
